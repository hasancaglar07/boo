## 📋 Project Summary

**Book Generator** - Multi-language AI publishing studio. From idea to publication-ready book in a single flow.

### One-Sentence Definition
A guided AI publishing studio that enables experts, educators, and content creators to produce multi-language, branded, publication-ready books from a single idea.

### Pricing Model
- **Starter** $19/month - 1 book/month
- **Creator** $39/month - 30 books/month  
- **Pro** $79/month - 80 books/month
- **Premium** $4 one-time - full access to a single book

### Tech Stack
- **Frontend:** Next.js 15, React, TypeScript, Tailwind CSS
- **Backend:** Python (FastAPI), Docker
- **Auth:** NextAuth.js
- **Payment:** Stripe
- **Database:** PostgreSQL (Prisma ORM)
- **AI:** Multi-provider (OpenAI, Gemini, Groq, Codefast)
- **Deployment:** Docker, Nginx

## 🧠 Core Concepts

### Funnel Flow
```
Landing → Start → Wizard → Generate → Signup → Preview → Paywall → Checkout → Export
```

### User Roles
- **Guest Builder** - Uses wizard without signing up
- **Free User** - Can preview, limited access
- **Premium** - Full book + export access

### Key Terms
- **Preview** - Free 20% preview of the book
- **Draft** - Book with incomplete generation
- **Wizard** - 5-step guided flow
- **Generate** - AI generation phase
- **Outline** - Chapter plan / table of contents draft
